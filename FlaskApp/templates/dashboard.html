About the projects

All the projects are listed under the "List of Projects"-tab. Most of the projects are from different assignments for MAH University. Mandatory to submit a project is a description, tags and a link to the github project. Without this you wont be able to submit a project. The reason is without this the project will be lacking too information for someone else to grasp what they project / assignment is about.

How do I contribute?

In order to submit a new project all you need is an account. In the top left corner press "Sign up" and create an account. After that a "Create new project" button will appear under List of Projects
